VeriPura — Interactive Lean Canvas
AI-Powered Compliance Platform for Regulated Imports · Seed Stage · March 2026

This repository hosts the interactive Lean Startup Canvas for VeriPura's seed fundraise, deployed via GitHub Pages as a single-file static site.

Live site: https://thomasoberlin.github.io/VP-lean-canvas/


What This Is
A single-page investor-facing microsite presenting VeriPura's Lean Canvas in an interactive format. Designed to be shared as a link alongside the one-pager and SAFE term summary.

Features:

Interactive canvas grid — click any cell to open a detailed modal overlay
5-year financial snapshot with revenue bar chart and income statement table
CPI corridor heat map — 17 origin countries colour-coded by Corruption Perceptions Index score
Founder bios
Contact / data room request CTA
Print to PDF via browser (File → Print or the header button)


Contents
index.html    Single self-contained file — all HTML, CSS and JS inline.

Deployment
This site is deployed via GitHub Pages from the main branch, root directory.

To update:

Edit index.html
Commit and push to main
GitHub Pages redeploys automatically (usually within 60 seconds)

To enable GitHub Pages on a new fork:

Go to Settings → Pages
Source: Deploy from a branch
Branch: main · Folder: / (root)
Save
